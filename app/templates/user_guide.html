After you submit your request, please add the {{ token_symbol }} script hash to your wallet:

Using Aphelion Wallet:

  1. Open Aphelion Wallet.
  2. If not already on Testnet, navigate to "Settings". Under "NETWORK" in the "Preferences" section, make sure that it says "TestNet". If not, select TestNet.
  3. Navigate to "Assets".
  4. Click on the "+" symbol with the "Add token" label. Copy `{{ token_script_hash }}` and paste into the prompt. Then select "Add". This will add the {{ token_symbol }} token to your Aphelion Wallet on the NEO Testnet.

Using Neon wallet:

  1. Open Neon Wallet.
  2. Navigate to "Manage Neon settings".
  3. Select "Manage Tokens".
  4. Select "TestNet" in the dropdown menu under "Network: Running on ". Select "+ Add a new token". Then, paste `{{ token_script_hash }}` into the empty space with the "script hash" placeholder. Then, select "Save".
  5. Login to your wallet.
  6. Your {{ token_symbol }} token balance should appear under the "Token" and "Balance" table.

Using neo-python:

  1. Type `np-prompt` into a terminal. This will open the Neo-python prompt.
  2. Type `open wallet {/path/to/your/wallet}` and enter your password when prompted. This will open the specified wallet.
  3. Type `import token {{ token_script_hash }}`. This allows the {{ token_symbol }} token to be viewed inside your wallet.
  4. Type `wallet`. This will show you details about the contents of your wallet. You should see "{{ token_name }}" appear in the "tokens" section of your wallet. You should also see `"[{{ token_symbol }}]: 10000 "` under the "synced_balances" section.
